# 接力送 · 开点确认流程

## 一、点位确认

- [ ] 确认需要开设的点位名称及地址
- [ ] 明确接力送摆放位置（大堂/电梯口/指定区域）
- [ ] 圈定该点位配送覆盖的楼栋范围

### 输出物

> 点位地图或平面图，标注：摆放点位置 + 送餐覆盖楼栋范围

---

## 二、物料筹备

| 物料 | 用途 | 备注 |
|------|------|------|
| KT板 | 点位标识，含二维码 | 附带点位二维码，骑手扫码交接 |
| 放餐篮子 | 骑手投餐暂存 | |
| 送餐篮子 | 上楼配送携带 | |
| 接力送马甲 | 人员身份标识 | |
| 油性马克笔 | 餐品标注 | 写房号等 |

### 特殊天气额外准备

| 物料 | 用途 |
|------|------|
| 防水贴纸 | 雨天外卖小票淋湿后无法写字，用贴纸替代标注 |

---

## 三、人员储备

### 定编规则

- 按预估单量配人：**平均每20单准备1人**
- 参考产能基准：12单/h/人

### 人员录入要求

人员须提前提供以下信息：

| 信息 | 说明 |
|------|------|
| 姓名 | 真实姓名 |
| 手机号码 | 可联系 |
| 身份证号码 | 实名认证用 |

> ⚠️ 提交后 **24小时内** 通过系统验证方可上岗

---

## 四、时间线

```
点位确认  →  物料到位  →  人员录入（提前24h）  →  系统验证通过  →  开点运营
```
